# Dungeon Generator

My final submission for CMPM 146 @ UCSC, Spring' 25. This generator uses
backtracking & weighted random selection to generate TBOI styled dungeons pretty
quickly, a map with 25-35 rooms can be generated in ~4 seconds.

## Framework for CMPM 146 - Assignment 8

This framework is for assignment 8 of the class CMPM 146 - Game AI. It was
developed in Unity version 6000.0.23f, but should also work in other Unity
versions.
